## Requirements

The following software versions (or newer) are required for full compatibility:

```conf
hyprland >= 0.51.1
kitty    >= 0.43.1
starship >= 1.21.1
nvim     >= 0.12.0 (Neovim)
code     >= 1.116.0 (Visual Studio Code)
waybar   >= 0.13.0
wofi     >= 1.5.3
rofi    >= 1.7.9
ranger   >= 1.9.4
hyprlock >= 0.9.2
neardfont >= 3.0.0
```
